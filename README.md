# Personal-Foreign-Words-Dictionary

# Персональный словарь иностранных слов

Мобильное Android-приложение для изучения иностранных языков с использованием алгоритма интервальных повторений (Spaced Repetition System).
Аналог Anki, но проще, полностью офлайн и с минималистичным интерфейсом.

## Особенности

- Создание колод и карточек (иностранное слово + перевод + пример + теги)
- Алгоритм SM-2 для автоматического планирования повторений
- Оценка запоминания («Легко», «Нормально», «Трудно», «Забыл»)
- Полностью офлайн-работа (Room Database)
- Удобный и минималистичный интерфейс на Jetpack Compose
- CRUD-операции над карточками и колодами
- Поиск и фильтрация по тегам
- Сброс прогресса колоды

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **Архитектура**: MVVM
- **База данных**: Room Persistence Library
- **Система сборки**: Gradle (KTS)
- **Тестирование**: JUnit 5


## Установка и запуск

1. Склонируйте репозиторий  
   ```bash
   git clone https://github.com/yourusername/personal-vocabulary.git
   ```
2. Откройте проект в Android Studio (Giraffe или новее)
3. Запустите на эмуляторе или реальном устройстве (min SDK 24)
